# ClearLearn AI

AI-powered microlearning platform designed to make education more accessible and engaging.

## Features
- AI-generated lessons
- Interactive quizzes
- AI answer verification
- Text-to-speech support
- Dyslexia-friendly mode
- High contrast mode

## Tech Stack
- Gemini API
- FastAPI
- Python
- HTML
- CSS
- JavaScript
